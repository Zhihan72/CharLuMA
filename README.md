# CharLuMA
This repository is provided for the double-blind reviewing process of ARR 2026 January Cycle. Redistribution of any data or code contained here is strictly prohibited during the review period.

We release the model architecture codes and training scripts of CharLuMA (under the train folder). The model architecture is built on LLaVA codebase.

We display a subset of Chart2NCode test set (under the image folder). Due to space constraints and the restriction on external links, the full Chart2NCode dataset cannot be included in this repository.
